## ✅ 正确填写步骤

1. **Key 字段**：
   - 删除当前的 "Name"
   - 输入：`VITE_LOGO_URL`

2. **Value 字段**：
   - 删除当前的 "VITE_LOGO_URL"
   - 输入完整的 URL：
     ```
     https://vxoacbydmzmjvnhvwjli.supabase.co/storage/v1/object/public/company%20Infomation/NEW%20LOGO-C.png
     ```

3. **Environments（环境）**：
   - 点击下拉菜单
   - 选择：**All Environments**（包含 Production, Preview, Development）

4. **Branch（分支）**：
   - 通常可以跳过，使用默认设置

5. **保存**：
   - 点击 **Save** 或 **Add** 按钮

## 🔄 如果已经保存错误的值

如果已经保存了错误的值，可以：
1. 在环境变量列表中找到错误的条目
2. 点击编辑（通常是个编辑图标）
3. 修改 Key 和 Value 为正确的值
4. 保存

或者：
1. 删除错误的条目
2. 重新添加，填写正确的值
