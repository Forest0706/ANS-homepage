# ⚡ 快速开始指南

## 📋 当前状态

✅ 项目已配置完成，可以直接使用！

## 🚀 三步快速部署

### 第一步：安装依赖

```bash
npm install
```

### 第二步：本地测试

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000，确认网站正常运行。

### 第三步：部署到 Vercel

#### 方法 A：通过命令行（推荐）

```bash
# 安装 Vercel CLI（如果还没安装）
npm i -g vercel

# 登录 Vercel
vercel login

# 部署到生产环境
vercel --prod
```

#### 方法 B：通过 GitHub（推荐生产环境）

1. **推送代码到 GitHub**
```bash
git add .
git commit -m "feat: 初始版本 - ANS官网"
git branch -M main
git remote add origin <你的GitHub仓库地址>
git push -u origin main
```

2. **在 Vercel 网站连接**
   - 访问 https://vercel.com
   - 点击 "Import Project"
   - 选择你的 GitHub 仓库
   - 点击 "Deploy"

以后每次 `git push` 都会自动部署！✨


### 3. Supabase 配置（如需要）

如果将来需要连接 Supabase：

1. 在 Supabase 创建项目
2. 创建 `.env.local` 文件（已加入 .gitignore）
3. 添加环境变量：
   ```
   VITE_SUPABASE_URL=你的URL
   VITE_SUPABASE_ANON_KEY=你的Key
   ```
4. 在 Vercel 控制台也添加相同的环境变量

## 🆘 遇到问题？

1. **构建失败**：检查 Node.js 版本（需要 18+）
2. **依赖安装失败**：删除 `node_modules` 和 `package-lock.json`，重新 `npm install`
3. **Vercel 部署失败**：检查 `vercel.json` 配置
4. **Git 推送失败**：确认远程仓库地址正确
